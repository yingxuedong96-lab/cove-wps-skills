# GB/T 9704-2012 国标公文格式排版

## 规范依据

GB/T 9704-2012《党政机关公文格式》国家标准。

## 四种公文格式

按标准实现时，可以按 4 类处理：1 种通用公文格式 + 第 10 章规定的 3 种特定格式（信函、命令〈令〉、纪要）。

### 格式类型对比

| 特征 | 通用公文 | 信函格式 | 命令(令)格式 | 纪要格式 |
|------|---------|---------|-------------|---------|
| 标准条款 | 第7章 | 10.1 | 10.2 | 10.3 |
| 适用文种 | 通知、决定、通报等 | 函、复函 | 命令、令 | 会议纪要 |
| 发文机关标志 | 含"文件"，距版心35mm | 不含"文件"，距上页边30mm | 含"命令/令"，距版心20mm | 含"纪要"，距版心35mm |
| 分隔线 | 红色单线 | 红色双线(上粗下细) | 无红色分隔线 | 无红色分隔线 |
| 发文字号 | 居中 | 居右上角顶格 | 无，改为令号居中 | 居中或无 |
| 首页页码 | 显示 | 不显示 | 显示 | 显示 |
| 特殊要素 | 无 | 无 | 签发人签名章 | 出席/请假/列席名单 |

### 格式识别规则

| 格式类型 | 识别规则 |
|---------|---------|
| 命令(令)格式 | 发文机关标志含"命令"或"令"字 |
| 纪要格式 | 发文机关标志含"纪要"字样 |
| 信函格式 | 发文字号右对齐，或发文机关标志不含"文件"且标题含"函" |
| 通用公文 | 默认（以上条件均不满足时） |

├────────────────────────────────────────────────┤

├────────────────────────────────────────────────┤

## 页码格式

- 字体：4号半角宋体阿拉伯数字
- 格式：一字线 + 页码 + 一字线（如 "— 1 —"）
- 位置：版心下边缘之下 7mm
- 单页码：居右空一字
- 双页码：居左空一字
- 首页：通用公文/命令/纪要显示，信函格式不显示
- 版记页：仅在版记页前有空白页时，空白页和版记页均不编排页码

## 元素识别规则

### 版头区域

| 元素 | 识别规则 |
|------|---------|
| 份号 | 6位纯数字，首段落 |
| 密级 | 含"秘密""机密""绝密" |
| 紧急程度 | 含"特急""加急" |
| 发文机关标志 | 含"文件"或机关名（政府、办公室等），居中 |
| 发文字号 | 匹配 `〔YYYY〕XX号` 格式 |
| 签发人 | 以"签发人："开头 |
| 令号 | 匹配 `第X号` 格式（命令格式专用） |

### 主体区域

| 元素 | 识别规则 |
|------|---------|
| 标题 | 分隔线后首个非空段落，≤50字 |
| 主送机关 | 以冒号结尾，含机关名 |
| 一级标题 | 以"一、""二、"等开头 |
| 二级标题 | 以"（一）""（二）"等开头 |
| 三级标题 | 以"1.""2."等数字点开头 |
| 四级标题 | 以"（1）""（2）"等开头 |
| 附件说明 | 以"附件："开头 |
| 成文日期 | 匹配 `YYYY年MM月DD日` 格式 |
| 落款 | 文末段落，无标点结尾或机构名 |
| 出席名单 | 以"出席：""请假：""列席："开头（纪要格式专用） |

### 版记区域

| 元素 | 识别规则 |
|------|---------|
| 抄送机关 | 以"抄送："开头 |
| 印发机关 | 含"印发"，不含冒号 |
| 印发日期 | 含日期格式 |

## 异常处理

本 Skill 严格遵循 GB/T 9704-2012《党政机关公文格式》国家标准，支持四种公文格式的自动识别与排版。适用于各级党政机关制发的公文。其他机关和单位的公文可以参照执行。
