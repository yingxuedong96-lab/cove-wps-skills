# 会议纪要红头文件排版

## 规范依据

会议纪要红头文件排版格式规范。

## 文档结构

```
┌────────────────────────────┐
│  红头（发文单位名称）        │  方正小标宋 42pt 红色 居中
│  ────────────────────────  │  红色横线
│  专题会议纪要               │  方正小标宋 42pt 红色 居中
│  〔2024〕第X号              │  仿宋 16pt 居中
├────────────────────────────┤
│  标题                       │  方正小标宋 22pt 居中
├────────────────────────────┤
│  一、XXX                    │  一级标题 黑体 16pt
│  （一）XXX                  │  二级标题 楷体 16pt
│  正文内容...                │  正文 仿宋 16pt 首行缩进
│  参会人员：XXX              │  参会人员 仿宋 16pt
├────────────────────────────┤
│  ────────────────────────  │  黑色横线
│  抄送：XXX                  │  抄送 仿宋 14pt
│  XXX印发                    │  印发 仿宋 14pt
│  共印X份                    │  共印 仿宋 14pt
└────────────────────────────┘
```

## 执行步骤

### Step 1 — 创建样式

调用 `executeFile`：

```text
filePath: "skills/REDformat/scripts/setup-styles.js"
params: {}
```

返回值：`{ success: boolean, message: string }`

确保文档中存在所需的会议纪要样式。

### Step 2 — 应用排版

调用 `executeFile`：

```text
filePath: "skills/REDformat/scripts/apply-format.js"
params: {}
```

返回值：`{ success: boolean, paragraphCount: number, appliedStyles: object }`

自动识别段落类型并应用对应样式：

| 元素类型 | 样式名 | 字体 | 字号 | 其他 |
|---------|--------|------|------|------|
| 红头 | 会议纪要-红头 | 方正小标宋简体 | 42pt | 红色，居中，加粗 |
| 公文标题字 | 会议纪要-标题字 | 方正小标宋简体 | 42pt | 红色，居中，加粗 |
| 发文字号 | 会议纪要-发文字号 | 仿宋_GB2312 | 16pt | 居中 |
| 标题 | 会议纪要-标题 | 方正小标宋简体 | 22pt | 居中，加粗 |
| 一级标题 | 会议纪要-一级标题 | 黑体 | 16pt | 左对齐 |
| 二级标题 | 会议纪要-二级标题 | 楷体_GB2312 | 16pt | 左对齐 |
| 正文 | 会议纪要-正文 | 仿宋_GB2312 | 16pt | 首行缩进2字符 |
| 参会人员 | 会议纪要-参会人员 | 仿宋_GB2312 | 16pt | 首行缩进 |
| 抄送 | 会议纪要-抄送 | 仿宋_GB2312 | 14pt | 左对齐 |
| 印发 | 会议纪要-印发 | 仿宋_GB2312 | 14pt | 左对齐 |
| 共印 | 会议纪要-共印 | 仿宋_GB2312 | 14pt | 左对齐 |

**元素识别规则**：

| 类型 | 识别规则 | 示例 |
|------|---------|------|
| 红头 | 第一段落 | "XX市人民政府" |
| 标题字 | 等于"专题会议纪要" | "专题会议纪要" |
| 发文字号 | 以〔〕或[]包裹 | "〔2024〕第15号" |
| 标题 | 发文字号后的段落，含"关于" | "关于XX项目的专题会议纪要" |
| 一级标题 | 以"一、""二、"等开头 | "一、会议背景" |
| 二级标题 | 以"（一）""（二）"等开头 | "（一）项目概述" |
| 参会人员 | 以"参会人员："开头 | "参会人员：张三、李四" |
| 抄送 | 以"抄送："开头 | "抄送：各相关部门" |
| 印发 | 以"印发"结尾 | "XX办公室印发" |
| 共印 | 匹配"共印X份" | "共印5份" |

**横线设置**：
- 红头下方：红色横线
- 抄送上方：黑色横线
- 印发上方：黑色横线

**页面设置**：A4纸张，标准页边距。

告知用户：排版完成，共处理 {paragraphCount} 个段落，应用样式：{appliedStyles}。

## 异常处理

**脚本报错时，回复：**

> 排版失败：{error}。请确认文档已打开后重试。
