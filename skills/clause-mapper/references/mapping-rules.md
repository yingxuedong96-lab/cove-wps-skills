# 条款映射规则库

## 一、映射模式总览

本规则库定义了草稿条款到标准条款的 5 种映射模式：

| 模式 | 说明 | 示例 |
|------|------|------|
| **一对一映射** | 单条款直接对应 | 投资款用途 → 第三条 |
| **多对一映射** | 多条合并到一条 | 反稀释权+优先认购权+... → 第七条 |
| **一对多映射** | 一条拆分到多条 | 常规条款 → 第十条~第十五条 |
| **整合映射** | 多条整合到前言 | 投资标的+创始股东+投资人 → 前言 |
| **拆分映射** | 不同条款合并 | 董事会安排+重大事项保护 → 第六条 |

| 回购条款 / 回购权 | 第八条 回购条款 | 直接映射 |
| 清算优先权 | 第九条 清算优先权 | 直接映射 |
| 增资款支付 / 交割 | 第五条 增资款支付与交割 | 语义相关 |
| 陈述与保证 / 声明保证 | 第十条 陈述与保证 | 语义等价 |
| 承诺 / 进一步承诺 | 第十一条 进一步承诺 | 语义相关 |

| 保密 | 第十三条 保密 | 直接映射 |
| 争议解决 | 第十四条 争议解决 | 直接映射 |

### 2. 多对一映射

| 草稿关键词（多个） | 标准条款 | 映射说明 |
|------------------|---------|---------|
| 反稀释权 | 第七条第1款 | 投资人特别权利 |
| 优先认购权 | 第七条第2款 | 投资人特别权利 |
| 优先购买权 | 第七条第3款 | 投资人特别权利 |
| 共售权 | 第七条第4款 | 投资人特别权利 |
| 领售权 | 第七条第5款 | 投资人特别权利 |
| 信息权 | 第七条第6款 | 投资人特别权利 |
| **注意**：以上所有权利合并到 **第七条 投资人特别权利** |

### 3. 整合映射

以下信息整合到 **协议前言** 部分：

| 草稿内容 | 标准位置 | 整合方式 |
|---------|---------|---------|
| 项目名称 | 前言 | 作为协议标题 |
| 签署日期 | 前言 | 协议签署时间 |
| 投资标的 / 公司名称 | 前言 | 定义"公司" |
| 创始股东 / 控股股东 / 实控人 | 前言 | 定义"创始股东" |
| 投资人 | 前言 | 定义"投资人A"、"投资人B" |
| 投资额 | 第二条 | 列明各方投资金额 |
| 投前估值 | 第二条 | 作为股权结构依据 |

### 4. 拆分映射

| 草稿条款 | 拆分到标准条款 |
|---------|--------------|
| 常规条款（知识产权、竞业禁止、保密、排他、费用、争议解决） | 第十条（陈述保证）、第十一条（承诺）、第十二条（违约）、第十三条（保密）、第十四条（争议解决）、第十五条（其他） |

### 5. 合并映射

| 草稿条款 | 合并到标准条款 |
|---------|--------------|
| 董事会安排 | 第六条第1、2款 |
| 重大事项保护 | 第六条第3款 |

## 四、语义理解规则

识别语义等价表述：

| 草稿表述 | 标准表述 | 判定依据 |
|---------|---------|---------|
| 交割前提条件 | 交割先决条件 | "前提"与"先决"语义等价 |
| 增资款用途 | 投资款用途 | "增资"与"投资"在融资场景下等价 |
| 声明与保证 | 陈述与保证 | 法律术语通用 |
| 董事会组成 | 董事会安排 | 语义相关 |

## 五、特殊处理规则

### 1. 未明确条款处理

当草稿条款无法直接匹配时：
1. 分析条款主题（投资、治理、权利、退出）
2. 根据主题归类到对应标准条款
3. 无法归类的，放入"第十五条 其他"

### 2. 多投资人处理

当存在多个投资人时：
1. 在前言中分别定义"投资人A"、"投资人B"
2. 在第二条中分别列明投资金额
3. 在权利条款中统一表述为"投资人"

### 3. 条款缺失处理

当标准条款所需的草稿内容缺失时：
1. 保留标准条款框架
2. 使用通用表述填充（如"具体金额以实际约定为准"）
3. 标注"需补充"提醒人工审核

## 六、映射示例

### 示例 1：一对一映射

**草稿条款**：
```
6. 投资款用途
用于主营业务研发、产品开发及团队建设。
```

**映射结果**：
```
第三条 增资款用途
公司应将本次投资款主要用于主营业务研发、产品开发及团队建设，不得挪作与主营业务无关之用途。
```

**映射类型**：一对一映射

---

### 示例 2：多对一映射

**草稿条款**：
```
9. 反稀释权
若公司后续以低于本轮价格发行任何股权或可转证券，则投资人享有基于狭义加权平均原则的反稀释保护。

10. 优先认购权
投资人享有后续融资的优先认购权。

11. 优先购买权 / 共售权 / 领售权
投资人享有优先购买权、共售权；在满足约定条件时享有领售权。

14. 信息权
投资人享有季度、半年度、年度财务报表及经营情况获取权，并可在合理范围内行使审计权。
```

**映射结果**：
```
第七条 投资人特别权利
（1）反稀释权：后续融资价格低于本轮价格时，投资人享有基于狭义加权平均原则的反稀释保护；
（2）优先认购权：投资人有权优先认购公司后续发行的股权或证券；
（3）优先购买权：创始股东或其他股东转让股权时，投资人享有优先购买权；
（4）共售权：创始股东对外转让股权时，投资人有权按同等条件共同出售；
（5）领售权：在约定条件满足时，投资人有权要求其他股东共同出售其持有股权；
（6）信息权：投资人有权取得公司季度、半年度、年度财务报表及经营资料，并在合理范围内进行审计。
```

**映射类型**：多对一映射

---

### 示例 3：整合映射

**草稿条款**：
```
项目名称：测试科技有限公司 A 轮融资
签署日期：2026 年 3 月
1. 投资标的
测试科技有限公司。
2. 创始股东、控股股东、实际控制人
张某、李某。
3. 投资人
投资机构A及其指定投资主体。
```

**映射结果**：
```
本《增资协议》由以下各方于 2026 年 3 月在中国签署：
测试科技有限公司（"公司"）；
张某、李某（合称"创始股东"）；
投资机构A指定投资主体（"投资人A"）；
跟投方B指定投资主体（"投资人B"）。
```

**映射类型**：整合映射
