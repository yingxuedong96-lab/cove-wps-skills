# 标准条款库

本文档包含增资协议的 15 条标准条款模板，供智能映射使用。

---

## 协议前言模板

```
本《增资协议》由以下各方于 {签署日期} 在中国签署：
{公司全称}（"公司"）；
{创始股东姓名}（合称"创始股东"）；
{投资人A名称}指定投资主体（"投资人A"）；
{投资人B名称}指定投资主体（"投资人B"）。
```

---

## 第一条 定义

```
第一条 定义
除非上下文另有说明，本协议中相关术语的含义以本协议及股东协议中的定义为准。
```

---

## 第二条 本次交易安排

```
第二条 本次交易安排
{投资人A名称}同意以人民币 {投资人A投资金额} 认购公司新增注册资本；
{投资人B名称}同意以人民币 {投资人B投资金额} 认购公司新增注册资本；
各方同意，本次投资中新增注册资本的一部分计入公司注册资本，其余计入资本公积；
本次增资完成后，公司注册资本、资本公积及股权结构按附件约定进行调整。
```

**参数提取**：
- 投资人名称：从草稿"投资人"条款提取
- 投资金额：从草稿"投资额"条款提取

---

## 第三条 增资款用途

```
第三条 增资款用途
公司应将本次投资款主要用于{投资款用途}，不得挪作与主营业务无关之用途。
```

**参数提取**：
- 投资款用途：从草稿"投资款用途"条款提取

---

## 第四条 交割先决条件

```
第四条 交割先决条件
投资人支付增资款应以下列条件全部满足或被豁免为前提：
1. 投资人完成满意的财务、法律及业务尽职调查；
2. 公司及创始股东作出的声明、保证持续真实、准确、完整；
3. 各方已签署正式交易文件，包括本协议及股东协议；
4. 公司已完成内部决议和外部审批/备案程序；
5. 自 TS 签署日至交割日，公司未发生重大不利变化。
```

---

## 第五条 增资款支付与交割

```
第五条 增资款支付与交割
在交割先决条件满足后，投资人应于约定期限内向公司支付增资款；
公司应向投资人出具缴款通知、出资证明书、更新后的股东名册及其他交割文件；
公司及创始股东应完成本次增资对应的工商变更登记。
```

---

## 第六条 公司治理安排

```
第六条 公司治理安排
公司董事会由 {董事会人数} 名董事组成；
投资人有权委派 {投资人董事席位} 名董事；
涉及{重大事项范围}等重大事项，应取得投资人委派董事同意。
```

**参数提取**：
- 董事会人数：从草稿"董事会安排"条款提取（默认5名）
- 投资人董事席位：从草稿"董事会安排"条款提取（默认1名）
- 重大事项范围：从草稿"重大事项保护"条款提取

---

## 第七条 投资人特别权利

```
第七条 投资人特别权利
（1）反稀释权：后续融资价格低于本轮价格时，投资人享有基于{反稀释方式}原则的反稀释保护；
（2）优先认购权：投资人有权优先认购公司后续发行的股权或证券；
（3）优先购买权：创始股东或其他股东转让股权时，投资人享有优先购买权；
（4）共售权：创始股东对外转让股权时，投资人有权按同等条件共同出售；
（5）领售权：在约定条件满足时，投资人有权要求其他股东共同出售其持有股权；
（6）信息权：投资人有权取得公司季度、半年度、年度财务报表及经营资料，并在合理范围内进行审计。
```

**参数提取**：
- 反稀释方式：从草稿"反稀释权"条款提取（默认"狭义加权平均"）

**映射说明**：
此条款由多个草稿条款合并而来：
- 草稿"反稀释权" → 第七条第1款
- 草稿"优先认购权" → 第七条第2款
- 草稿"优先购买权" → 第七条第3款
- 草稿"共售权" → 第七条第4款
- 草稿"领售权" → 第七条第5款
- 草稿"信息权" → 第七条第6款

---

## 第八条 回购条款

```
第八条 回购条款
若交割后 {回购期限} 年内公司未完成合格 IPO，或公司/创始股东严重违反交易文件，投资人有权要求公司或创始股东回购投资人持有股权。回购价格为：原投资额 + 每年 {回购利率} 单利 + 已宣告未支付股息。
```

**参数提取**：
- 回购期限：从草稿"回购条款"提取（默认8年）
- 回购利率：从草稿"回购条款"提取（默认10%）

---

## 第九条 清算优先权

```
第九条 清算优先权
发生清算、出售绝大多数资产或导致控制权变化的并购等清算事件时，投资人有权优先于普通股股东取得其 {清算优先比例} 投资成本及已宣告未支付股息；剩余资产由全体股东按实缴出资比例分配。
```

**参数提取**：
- 清算优先比例：从草稿"清算优先权"提取（默认100%）

---

## 第十条 陈述与保证

```
第十条 陈述与保证
公司及创始股东应就公司的设立与存续、股权结构、财务状况、资产、合同、知识产权、诉讼、合规经营、员工与社保公积金缴纳等事项作出真实、准确、完整的陈述与保证；投资人亦应就其主体资格、签约能力及资金来源合法性作出陈述与保证。
```

---

## 第十一条 进一步承诺

```
第十一条 进一步承诺
公司及创始股东承诺：
1. 在交割日前维持正常经营，不进行异常交易或重大不利变更；
2. 配合投资人尽调并提供必要资料；
3. 保障投资人交割后依法享有股东权利；
4. 在交割后依法合规经营，建立内控、财务及知识产权管理制度。
```

---

## 第十二条 违约责任

```
第十二条 违约责任
任何一方违反本协议项下陈述、保证、承诺或义务的，应赔偿守约方因此遭受的全部损失，包括直接损失、间接损失及合理维权费用。
```

---

## 第十三条 保密

```
第十三条 保密
各方应就本次交易、交易文件及获悉的其他方保密信息承担保密义务，法律法规或监管要求另有规定的除外。
```

---

## 第十四条 争议解决

```
第十四条 争议解决
因本协议引起的或与本协议有关的争议，应提交中国国际经济贸易仲裁委员会在北京仲裁解决。
```

---

## 第十五条 其他

```
第十五条 其他
本协议自各方签署之日起生效。未尽事宜，由各方另行协商并签署补充协议。
```

## 条款使用说明

### 1. 参数化字段

以下字段需要从草稿条款中智能提取并填充：

- `{签署日期}` - 从草稿"签署日期"提取
- `{公司全称}` - 从草稿"投资标的"提取
- `{创始股东姓名}` - 从草稿"创始股东"提取
- `{投资人A名称}`、`{投资人B名称}` - 从草稿"投资人"提取
- `{投资人A投资金额}`、`{投资人B投资金额}` - 从草稿"投资额"提取
- `{投资款用途}` - 从草稿"投资款用途"提取
- `{董事会人数}` - 从草稿"董事会安排"提取
- `{投资人董事席位}` - 从草稿"董事会安排"提取
- `{重大事项范围}` - 从草稿"重大事项保护"提取
- `{反稀释方式}` - 从草稿"反稀释权"提取
- `{回购期限}` - 从草稿"回购条款"提取
- `{回购利率}` - 从草稿"回购条款"提取
- `{清算优先比例}` - 从草稿"清算优先权"提取

### 2. 条款完整性

根据实际融资场景，标准条款可能包括但不限于以上 15 条。以下条款为必备条款：
- 第一条（定义）
- 第二条（交易安排）
- 第三条（增资款用途）
- 第四条（交割先决条件）
- 第五条（支付与交割）
- 第六条（公司治理）
- 第七条（投资人权利）
- 第十三条（保密）
- 第十四条（争议解决）
- 第十五条（其他）

### 3. 条款顺序

标准条款应按以下顺序排列：
1. 前言（整合映射部分）
2. 第一条 ~ 第十五条（按编号顺序）

### 4. 特殊条款处理

- **定义条款**：如草稿中没有明确定义，保留标准模板
- **附件**：股权结构、估值计算等建议作为附件，不在正文中详细展开
- **常规条款**：如草稿中的"知识产权"、"竞业禁止"等，可整合到第十条或第十一条，或单独添加补充条款
