# 增资协议条款模板

## 概述
本文档提供《增资协议》15条条款的标准表述。用户原文没有的内容，使用本模板的表述。

---

## 协议前言

```
本《增资协议》由以下各方于【签署日期】在【签署地点】签署：
【公司名称】（"公司"）；
【创始股东名单】（合称"创始股东"）；
【投资人A名称】指定投资主体（"投资人A"）；
【投资人B名称】指定投资主体（"投资人B"）〔如有〕。
```

---

## 第一条 定义

```
除非上下文另有说明，本协议中相关术语的含义以本协议及股东协议中的定义为准。
```

---

## 第二条 本次交易安排

```
投资人A同意以人民币【金额】万元认购公司新增注册资本；〔投资人B同意以人民币【金额】万元认购公司新增注册资本；〕各方同意，本次投资中新增注册资本的一部分计入公司注册资本，其余计入资本公积；本次增资完成后，公司注册资本、资本公积及股权结构按附件约定进行调整。
```

---

## 第三条 增资款用途

```
公司应将本次投资款主要用于【用途描述】，不得挪作与主营业务无关之用途。
```

---

## 第四条 交割先决条件

```
投资人支付增资款应以下列条件全部满足或被豁免为前提：
1. 投资人完成满意的财务、法律及业务尽职调查；
2. 公司及创始股东作出的声明、保证持续真实、准确、完整；
3. 各方已签署正式交易文件，包括本协议及股东协议；
4. 公司已完成内部决议和外部审批/备案程序；
5. 自 TS 签署日至交割日，公司未发生重大不利变化。
```

---

## 第五条 增资款支付与交割

```
在交割先决条件满足后，投资人应于约定期限内向公司支付增资款；公司应向投资人出具缴款通知、出资证明书、更新后的股东名册及其他交割文件；公司及创始股东应完成本次增资对应的工商变更登记。
```

---

## 第六条 公司治理安排

```
公司董事会由【数量】名董事组成；投资人有权委派【数量】名董事；涉及章程修改、注册资本变化、优先级证券发行、分红、重大借款、重大投资、重大资产处置、关联交易、高管任免等重大事项，应取得投资人委派董事同意。
```

---

## 第七条 投资人特别权利

```
反稀释权：后续融资价格低于本轮价格时，投资人享有基于【狭义加权平均/全额棘轮】原则的反稀释保护；
优先认购权：投资人有权优先认购公司后续发行的股权或证券；
优先购买权：创始股东或其他股东转让股权时，投资人享有优先购买权；
共售权：创始股东对外转让股权时，投资人有权按同等条件共同出售；
领售权：在约定条件满足时，投资人有权要求其他股东共同出售其持有股权；
信息权：投资人有权取得公司季度、半年度、年度财务报表及经营资料，并在合理范围内进行审计。
```

---

## 第八条 回购条款

```
若交割后【年限】年内公司未完成合格 IPO，或公司/创始股东严重违反交易文件，投资人有权要求公司或创始股东回购投资人持有股权。回购价格为：原投资额 + 每年【利率】%单利 + 已宣告未支付股息。
```

---

## 第九条 清算优先权

```
发生清算、出售绝大多数资产或导致控制权变化的并购等清算事件时，投资人有权优先于普通股股东取得其【比例】%投资成本及已宣告未支付股息；剩余资产由全体股东按实缴出资比例分配。
```

---

## 第十条 陈述与保证

```
公司及创始股东应就公司的设立与存续、股权结构、财务状况、资产、合同、知识产权、诉讼、合规经营、员工与社保公积金缴纳等事项作出真实、准确、完整的陈述与保证；投资人亦应就其主体资格、签约能力及资金来源合法性作出陈述与保证。
```

---

## 第十一条 进一步承诺

```
公司及创始股东承诺：
1. 在交割日前维持正常经营，不进行异常交易或重大不利变更；
2. 配合投资人尽调并提供必要资料；
3. 保障投资人交割后依法享有股东权利；
4. 在交割后依法合规经营，建立内控、财务及知识产权管理制度。
```

---

## 第十二条 违约责任

```
任何一方违反本协议项下陈述、保证、承诺或义务的，应赔偿守约方因此遭受的全部损失，包括直接损失、间接损失及合理维权费用。
```

---

## 第十三条 保密

```
各方应就本次交易、交易文件及获悉的其他方保密信息承担保密义务，法律法规或监管要求另有规定的除外。
```

---

## 第十四条 争议解决

```
因本协议引起的或与本协议有关的争议，应提交【仲裁机构名称】在【地点】仲裁解决。
```

---

## 第十五条 其他

```
本协议自各方签署之日起生效。未尽事宜，由各方另行协商并签署补充协议。
```
