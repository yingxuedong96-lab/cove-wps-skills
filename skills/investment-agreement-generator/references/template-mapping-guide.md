# 信息提取与映射指南

## 概述

本文档说明如何从投资条款清单中提取关键信息，用于生成增资协议。

## 必须提取的信息

| 字段名 | 含义 | 数据来源关键词 |
|--------|------|----------------|
| companyName | 目标公司全称 | 公司、目标公司、标的公司 |
| founderName | 创始人姓名 | 创始股东、创始人、实际控制人 |
| investorA | 投资人A的实体名称 | 投资人A、领投方 |
| signDate | 签署日期 | 签署日期、交割日、投资完成日 |
| **mainBusiness** | **主营业务/投资款用途** | **投资款用于、投资款用途、资金用途、主要用于、从事** |

## 可选提取的信息

| 字段名 | 含义 | 数据来源关键词 |
|--------|------|----------------|
| esopPlatform | 员工持股平台名称 | ESOP、员工持股平台、持股平台 |
| investorB | 投资人B的实体名称 | 投资人B、跟投方 |
| resourcePlatform | 资源方持股平台名称 | 资源方 |
| companyCode | 公司统一社会信用代码 | 统一社会信用代码 |
| founderIdType | 创始人证件类型 | 身份证、护照 |
| founderIdNo | 创始人证件号 | 身份证号、证件号 |
| founderNationality | 创始人国籍 | 国籍 |
| establishDate | 公司成立日期 | 成立于、成立日期 |
| registeredCapital | 注册资本 | 注册资本 |
| totalInvestment | 投资总额 | 投资总额、合计投资 |
| newCapital | 新增注册资本 | 新增注册资本 |

## 智能提取规则

### 1. 公司名称
- 关键词：公司、目标公司、标的公司
- 示例：条款中写"智慧山川科技有限公司"，提取为 companyName

### 2. 创始股东
- 关键词：创始股东、创始人、实际控制人
- 多人时用顿号分隔
- 示例：条款中写"李云、王大龙"，提取为 founderName

### 3. 投资人信息
- 关键词：投资人A、投资人B、领投方、跟投方
- 包含投资实体名称
- 示例：条款中写"慕云资本指定投资主体"，提取为 investorA

### 4. 日期信息
- 关键词：签署日、交割日、投资完成日
- 格式：XXXX年XX月XX日

### 5. 主营业务/投资款用途（重要）
**关键词：投资款用于、投资款用途、资金用途、主要用于、从事**

**识别模式：**
- "投资款主要用于XXX" → 提取XXX作为mainBusiness
- "投资款用途：XXX" → 提取XXX作为mainBusiness
- "资金将用于XXX" → 提取XXX作为mainBusiness
- "主要用于XXX的研发与制造" → 提取"XXX的研发与制造"

**示例：**
```
输入：公司应将本次投资款主要用于AI智能机器人的研发与制造
提取：mainBusiness = "AI智能机器人的研发与制造"
```

```
输入：投资款用途：新能源汽车核心零部件的研发和生产
提取：mainBusiness = "新能源汽车核心零部件的研发和生产"
```

## 映射示例

### 输入（投资条款清单片段）
```
本条款清单由以下各方签署：
公司：智慧山川科技有限公司
创始股东：李云、王大龙
投资人A：慕云资本指定投资主体
投资人B：灵犀资本指定投资主体

投资人A同意以人民币5000万元认购公司新增注册资本...
公司应将本次投资款主要用于AI智能机器人的研发与制造...
```

### 提取结果
```json
{
  "companyName": "智慧山川科技有限公司",
  "founderName": "李云、王大龙",
  "investorA": "慕云资本指定投资主体",
  "investorB": "灵犀资本指定投资主体",
  "signDate": "2026年6月",
  "mainBusiness": "AI智能机器人的研发与制造",
  "totalInvestment": "7500万元"
}
```

## 注意事项

1. **必填信息**：公司名称、创始股东、至少一个投资人
2. **投资款用途**：务必识别并映射到 mainBusiness 字段，用于鉴于条款第1条
3. **可选信息**：ESOP持股平台、资源方持股平台、投资人B
4. **格式要求**：日期统一使用中文格式（年月日）
5. **多人情况**：姓名之间使用顿号（、）分隔

## 鉴于条款映射说明

鉴于条款第1条中的【】需要映射 mainBusiness：
```
原文：集团公司（定义见下文）主要从事【】（"主营业务"...
映射：集团公司（定义见下文）主要从事AI智能机器人的研发与制造（"主营业务"...
```
